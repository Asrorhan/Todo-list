# 📝 Todo List

A simple and responsive Todo List web app built with **HTML, CSS, and JavaScript**.  
You can add, edit, and delete tasks easily.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- 📱 Responsive design
- 🎨 Clean UI with scroll support for long task lists

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
